Features

Multiple Instances

MCUnified allows you to have multiple minecraft instances to keep your gameplay organized.

One-Click Mod Installing

Using a special api, developed by us, we are able to get mods from curseforge.com and download them directly from the launcher.

Forge Installation

Using code based on the official forge installer, we are able to allow you to install forge with the click of a button.

MCLauncher-API at the Core

In the core of our launcher we use an api developed by Tomas Jasek You can view his code here. This launcher could not have been made without him
